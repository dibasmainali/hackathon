# CareerBridge Nepal - Job & Skill Matching Platform for Youth

> **Hackathon Project**: Empowering Nepali youth with career guidance, resume building, and job matching opportunities.

## ğŸ¯ Project Overview

CareerBridge Nepal is a comprehensive job and internship matching platform designed specifically for students and fresh graduates in Nepal. Our platform bridges the gap between talented youth seeking opportunities and companies looking for skilled individuals.

### Main Goals

1. **Career Guidance**: Provide personalized career advice and roadmap for unskilled individuals
2. **Resume Building**: Help users create professional resumes tailored to their career goals
3. **Job Matching**: Show relevant company postings and hiring opportunities to skilled candidates
4. **Company Platform**: Enable companies to post jobs and find the right talent

## ğŸš€ Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn
- MongoDB (for backend)

### Frontend Setup

```bash
cd frontend-folder
npm install
npm run dev
```

Open `http://localhost:5173` in your browser.

### Backend Setup

```bash
cd backend-folder
npm install
# Configure database in config/db.config.js
npm start
```

The backend will run on `http://localhost:5000` (or configured port).
